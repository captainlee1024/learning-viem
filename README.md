# Terry Wallet

## quick start

```
$node src/wallet_cli.js -h

#output
Usage: wallet_cli [options] [command]

Terry wallet CLI

Options:
  -V, --version          output the version number
  -h, --help             display help for command

Commands:
  gen-mnemonic           Generate a new mnemonic phrase
  import-mnemonic        Import mnemonic phrase, gen default account
  gen-account            Generate default account
  erc20-read [options]   Read data from an ERC20 contract
  erc20-write [options]  Write data to an ERC20 contract
  help [command]         display help for command
```
